# TradeFlow Wallet UI

Frontend prototype for the SIT blockchain transaction testing and asset tokenization framework.

## Features

- Blockchain wallet dashboard
- Portfolio tracking
- Transaction simulation
- Blockchain explorer
- Validator monitoring
- Automated test harness
- Dark/light mode

## Tech Stack

- React
- Vite
- React Router
- Recharts
- Lucide Icons

## Installation

```bash
npm install
npm run dev
```

## Future Integration

This frontend is designed to integrate with:
- Rust blockchain backend (`chain-rs`)
- Automated test harness
- Casper-style Proof-of-Stake simulation
- Keccak hashing validation
